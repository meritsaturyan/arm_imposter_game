# Arm Imposter — веб-версия (браузер)

Та же игра «Impostor» на армянском: один игрок — Impostor, остальные видят слово и по подсказкам и голосованию должны его вычислить.

## Как открыть игру в браузере

### Вариант 1: Локальный сервер (рекомендуется)

Чтобы клиент мог посмотреть игру у себя, запустите в папке **web** простой сервер и откройте в браузере указанный адрес (или отправьте ссылку клиенту в одной сети):

```bash
cd web
python3 -m http.server 8080
```

Откройте в браузере: **http://localhost:8080**  
Чтобы клиент зашёл с другого устройства в той же сети: **http://ВАШ_IP:8080** (например `http://192.168.1.5:8080`).

### Вариант 2: Выложить в интернет

Скопируйте содержимое папки **web** (файлы `index.html`, `style.css`, `game.js` и папку `data/`) на любой хостинг:

- **GitHub Pages** — создайте репозиторий, загрузите папку web в ветку `main`, в Settings → Pages выберите source: main, папку `/ (root)`. Ссылка будет вида `https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПО/`.
- **Netlify** / **Vercel** — перетащите папку web в окно загрузки или подключите репозиторий.
- Любой другой хостинг статики — загрузите файлы так, чтобы при заходе на сайт открывался `index.html`.

После этого клиент сможет открыть игру по ссылке в любом браузере.

### Вариант 3: Открыть файл локально

Двойной клик по **index.html** или «Открыть файл» в браузере.  
Словарь подгружается из `data/words_hy.json`; если папка `data` рядом с `index.html`, всё будет работать. В некоторых браузерах при открытии через `file://` загрузка JSON может блокироваться — тогда используйте вариант 1 (локальный сервер).

## Управление

1. **Старт** — «Սկսել խաղը» → настройки (игроки, импосторы, таймер) → «Շարունակել» → выбор категории.
2. **Раунд** — по очереди показываете карточку (слово или «ԻՄՊՈՍՏՈՐ»), затем обсуждение и голосование.
3. Если голосованием вычислили импостора — победа команды, иначе победа импостора.

Файлы в **web**: `index.html`, `style.css`, `game.js`, `data/words_hy.json`.
