# Arm Imposter — веб-версия для GitHub Pages

Эта папка публикуется на **GitHub Pages**: в настройках репозитория (Settings → Pages) выберите источник **Deploy from a branch**, ветку **main**, папку **/docs**.

Игра откроется по ссылке: `https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПОЗИТОРИЯ/`

Файлы не меняйте вручную — при обновлении веб-версии синхронизируйте содержимое с папкой **web/** (index.html, style.css, game.js, data/words_hy.json).
